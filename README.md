# hashkit

High-performance hashing primitives shared across Santh crates.

- **FNV-1a** – stable, byte-order-independent 64-bit hashing.
- **SplitMix64** – high-quality seed finalization and compact pair hashing.
- **WyHash** – fast, deterministic bulk hashing of byte slices.
- **BLAKE3** – cryptographically secure, streaming content hashing for
  deduplication and content-addressed storage.
